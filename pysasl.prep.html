pysasl.prep Package¶

class pysasl.prep.Preparation(*args, **kwargs)[source]¶

Any callable that prepares a string value to improve the likelihood that comparisons behave in an expected manner.

pysasl.prep.default_prep(source)¶

Prepares the source string using the default preparation algorithm. This default is saslprep() if available otherwise noprep().
