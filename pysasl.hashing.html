pysasl.hashing Package¶

Provides an abstraction and several implementations for the ability to hash and verify secrets.

class pysasl.hashing.BuiltinHash(*, hash_name='sha256', salt_len=16, rounds=500000)[source]¶

Implements HashInterface using the hashlib.pbkdf2_hmac() function and random salt.

The constructor arguments are the values used when encoding. When decoding, these values are read from the digest format.

Parameters:

  • hash_name (Literal['sha1', 'sha256', 'sha512']) – The hash name.

  • salt_len (int) – The length of the random salt.

  • rounds (int) – The number of hash rounds.

pysasl.hashing.get_hash(*, passlib_config=None)[source]¶

Provide a secure, default HashInterface implementation.

If passlib_config is given, a CryptContext is loaded from it. Otherwise, the returned implementation depends on whether passlib is available.

If passlib is available, a CryptContext is created with a set of active hashes, defaulting to pbkdf2_sha256 for new digests.

If passlib is not available, a BuiltinHash with default settings is created. This is intended to be compatible with the passlib default if it is installed later.

Parameters:

passlib_config (Optional[str]) – A passlib config file.

Return type:

HashInterface

See also

passlib.context.CryptContext.from_path()
